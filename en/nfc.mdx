| --- | --- | --- |
| Forward | 5 | cfor |
| Left | 4 | clef |
| Right | 4 | crig |
| Backward | 1 | cbac |
| Repeat 2 | 2 | r002 |
| Repeat 3 | 1 | r003 |
| Repeat 4 | 1 | r004 |
| Repeat 5 | 1 | r005 |
| Repeat 6 | 1 | r006 |
| Repeat Random number (1 to 6) | 1 | rrnd |
| Function | 3 | cfun |
| Random Move | 1 | crnd |

## **How to Write Data to NFC tag**

You can do self token just write text code of to NFC token. You can do any numbers (from 000 to 999), any repeats and any numbers for arithmetic.

For writing text Data to NFC token please go to Android Play Store and install **RFID NFC Reader** or **NFC Tools** at your phone.
