## **Comment écrire des données sur une étiquette NFC**

Vous pouvez créer votre propre jeton — il suffit d’écrire un code texte sur l’étiquette NFC. Vous pouvez utiliser n’importe quels nombres (de 0 à 999), n’importe quelles répétitions et n’importe quels nombres pour des opérations arithmétiques.

Pour enregistrer des données textuelles sur une étiquette NFC, ouvrez le Google Play Store et installez **RFID NFC Reader** ou **NFC Tools** sur votre téléphone.

Le guide sur l’écriture d’une étiquette NFC se trouve ici — [youtu.be/UbKaQxZs8JA](https://youtu.be/UbKaQxZs8JA)
